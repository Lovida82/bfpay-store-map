# 비플페이 검증 가맹점 지도 - 사용자 매뉴얼

> **서비스 주소**: https://bppay-store-map.vercel.app/

---

## 1. 회원가입 / 로그인

### 회원가입
1. 우측 상단 **프로필 아이콘** 클릭
2. **회원가입** 선택
3. 이메일, 비밀번호, 닉네임 입력
4. **회원가입** 버튼 클릭
5. 입력한 이메일로 인증 메일 발송됨 → 메일 확인 후 인증 링크 클릭

### 로그인
1. 우측 상단 **프로필 아이콘** 클릭
2. 이메일, 비밀번호 입력
3. **로그인** 버튼 클릭

---

## 2. 가맹점 검색하기

### 지도에서 찾기
- 지도를 드래그하여 원하는 지역으로 이동
- 마커를 클릭하면 가맹점 정보 팝업 표시
- 팝업 클릭 시 상세 페이지로 이동

### 검색창 사용
- 좌측 패널(모바일: 하단)의 검색창에 **가게명** 또는 **주소** 입력
- 예: "홍콩반점", "평택", "강남역" 등

### 업종별 필터
- 지도 좌측 상단에서 업종 선택 (음식점, 카페, 편의점 등)

### 현재 지도 범위 조회
1. 원하는 지역으로 지도 이동
2. 좌측 상단 **조회** 버튼 클릭
3. 현재 화면에 보이는 가맹점만 리스트에 표시
4. 다시 전체 목록을 보려면 **전체 목록 보기** 클릭

---

## 3. 가맹점 등록하기

> 로그인 필요

### 방법 1: 수동 등록
1. 상단 **등록** 버튼 클릭
2. **수동 등록** 탭 선택
3. 가맹점 정보 입력
   - 가게명 (필수)
   - 주소 (필수) - 정확한 도로명/지번 주소 입력
   - 전화번호 (선택)
   - 업종 선택
4. **등록하기** 버튼 클릭

### 방법 2: 사진으로 등록 (OCR)
1. 상단 **등록** 버튼 클릭
2. **OCR 등록** 탭 선택
3. 가맹점 스티커나 간판 사진 업로드
4. AI가 자동으로 가게명, 주소 추출
5. 정보 확인 후 **등록하기** 클릭

### 방법 3: Excel 일괄 등록
1. 상단 **등록** 버튼 클릭
2. **Excel 등록** 탭 선택
3. **샘플 템플릿 다운로드** 클릭하여 양식 확인
4. Excel 파일 작성 (가게명, 주소 필수)
5. 파일 업로드 → 미리보기 확인
6. **일괄 등록** 버튼 클릭

**Excel 양식 예시:**
| 가게명 | 주소 | 전화번호 | 업종 |
|--------|------|----------|------|
| 홍콩반점 | 서울 강남구 테헤란로 123 | 02-1234-5678 | 음식점 |
| 스타벅스 강남점 | 서울 강남구 강남대로 456 | 02-2345-6789 | 카페 |

## 4. 결제 검증하기

> 로그인 필요

실제로 비플페이/제로페이 결제를 시도한 후 결과를 기록합니다.

1. 가맹점 상세 페이지 이동 (마커 클릭 → 팝업 클릭)
2. **검증하기** 버튼 클릭
3. 결제 결과 선택
   - ✅ **결제 성공**: 정상적으로 결제됨
   - ❌ **결제 실패**: 결제 거부, 단말기 없음 등
4. 코멘트 입력 (선택)
5. **검증 제출** 클릭

---

## 5. 후기 남기기

> 로그인 필요

1. 가맹점 상세 페이지 이동
2. **후기 작성** 버튼 클릭
3. 정보 입력
   - 별점 (1~5점)
   - 결제 성공 여부
   - 방문일
   - 후기 내용
4. **후기 등록** 클릭

---

## 6. 마이페이지

1. 우측 상단 **프로필 아이콘** 클릭
2. **마이페이지** 선택

### 확인 가능한 정보
- **내 정보**: 닉네임, 이메일, 가입일
- **활동 통계**: 등록한 가맹점 수, 검증 횟수
- **내가 등록한 가맹점**: 목록 확인 및 삭제
- **내 검증 내역**: 검증한 가맹점 기록

### 프로필 수정
1. **프로필 수정** 버튼 클릭
2. 닉네임 변경
3. **저장** 클릭

---

## 7. 마커 색상 안내

지도의 마커 색상은 가맹점의 신뢰도를 나타냅니다.

| 색상 | 의미 | 조건 |
|------|------|------|
| 🟢 녹색 | 검증 완료 | 신뢰도 70% 이상 + 검증 3회 이상 |
| 🟡 노란색 | 검증 중 | 신뢰도 40~70% |
| 🔴 빨간색 | 주의 | 신뢰도 40% 미만 |
| ⚫ 회색 | 미검증 | 아직 검증 기록 없음 |

**신뢰도 계산**: (결제 성공 건수 ÷ 전체 검증 건수) × 100

## 8. 모바일 사용법

- **하단 시트**: 화면 아래 "가맹점 목록" 탭을 위로 드래그하면 목록 펼쳐짐
- **지도 이동**: 손가락으로 드래그
- **확대/축소**: 두 손가락 핀치
- **현재 위치**: 우측 상단 위치 버튼 클릭

## 자주 묻는 질문 (FAQ)

**Q. 등록한 가맹점이 지도에 안 보여요**
- 주소가 정확한지 확인해주세요
- 새로고침(F5) 후 다시 확인해주세요

**Q. 검증은 여러 번 할 수 있나요?**
- 네, 같은 가맹점에 여러 번 검증 가능합니다
- 최신 검증 기록이 신뢰도에 반영됩니다

**Q. 잘못 등록한 가맹점은 어떻게 삭제하나요?**
- 마이페이지 → 내가 등록한 가맹점 → 삭제 버튼

**Q. 비밀번호를 잊어버렸어요**
- 로그인 화면에서 "비밀번호 찾기" 클릭
- 가입한 이메일로 재설정 링크 발송
